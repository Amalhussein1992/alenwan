# Episodes Views

## Views to Create:

1. **index.blade.php** - List all Episodes
2. **create.blade.php** - Form to create new Episode
3. **edit.blade.php** - Form to edit existing Episode

## Fields:

- `series_id`
- `title`
- `description`
- `season_number`
- `episode_number`
- `duration_minutes`
- `video_url`
- `thumbnail`
- `status`
