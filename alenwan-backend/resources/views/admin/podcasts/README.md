# Podcasts Views

## Views to Create:

1. **index.blade.php** - List all Podcasts
2. **create.blade.php** - Form to create new Podcast
3. **edit.blade.php** - Form to edit existing Podcast

## Fields:

- `title`
- `description`
- `host`
- `guests`
- `duration_seconds`
- `release_date`
- `cover_image`
- `audio_file`
- `status`

## Copy From:

You can copy and modify from: `resources/views/admin/movies/`

## Route:

URL: `/admin/podcasts`
