# Categories Views

## Views to Create:

1. **index.blade.php** - List all Categories
2. **create.blade.php** - Form to create new Category
3. **edit.blade.php** - Form to edit existing Category

- `description`
- `type`
- `sort_order`
- `is_active`
- `image`
